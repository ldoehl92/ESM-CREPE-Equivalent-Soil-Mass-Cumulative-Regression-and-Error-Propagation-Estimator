# ESM–CREPE: Equivalent Soil Mass - Cumulative Regression and Error Propagation Estimator

## Overview
ESM–CREPE is a R-based package for the purpose of estimating differences or changes in soil organic carbon (SOC) stocks between soil systems using the equivalent soil mass (ESM) method.

## Downloading the code
We suggest the following ways to download this repository:

1. Cloning the repository
You can clone this repository using git:

git clone https://github.com/ldoehl92/ESM-CREPE-Equivalent-Soil-Mass-Cumulative-Regression-and-Error-Propagation-Estimator/tree/main

2. Cloning the repository with option to contribute to project
If you are interested to contribute to changes and share your inputs to advancing the open source project via GitHub, you should install the gh CLI tool. Then clone the source code using:

gh repo fork ldoehl92/ESM-CREPE-Equivalent-Soil-Mass-Cumulative-Regression-and-Error-Propagation-Estimator/tree/main

3. Download as a zip
You can download the package as a zip file from the following link: https://github.com/ldoehl92/ESM-CREPE-Equivalent-Soil-Mass-Cumulative-Regression-and-Error-Propagation-Estimator/tree/main.zip
